情况二 - 存在则返回，无则返回屁

情况三 - 存在则返回，无则由函数产生

情况四 - 夺命连环null检查

由上述四种情况可以清楚地看到，Optional+Lambda可以让我们少写很多ifElse块。尤其是对于情况四那种夺命连环null检查，传统java的写法显得冗长难懂，而新的Optional+Lambda则清新脱俗，清楚简洁。
